# Python flask template

## Setup
To get started you will need to install the required packages for the python server.
These are in `requirements.txt` and can be downloaded with the following command
```
pip3 install -r requirements.txt
```

## Running flask
Now that the requirements are installed you can run the flask server with
```
flask --app src/main.py run
```
The server should then be accessible from `http://localhost:5000`
